# Claude Mobile Changes

This folder contains all mobile app changes created by Claude Code sessions.

## Structure

This mirrors the main `CashMonki/` app structure:

```
claude-mobile/
├── Authentication/     # Auth-related changes (login, register, etc.)
├── Pages/             # New or modified pages (HomePage, ReceiptsPage, SettingsPage)
├── Onboarding/        # Onboarding flow changes
├── Managers/          # State managers and business logic
├── Functions/         # Utility functions, services, models
├── Components/        # Reusable UI components
├── Sheets/            # Modal sheets and dialogs
├── DesignSystem/      # Design system updates, colors, fonts
├── Models/            # Data models and structures
└── Utils/             # Helper utilities and tools
```

## Usage

1. **New Features** - Created files go in the appropriate folder
2. **Modifications** - Modified versions of existing files (if needed for reference)
3. **Integration** - Review changes here, then merge into main `CashMonki/` folder when ready
